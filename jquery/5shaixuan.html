2、如果是一个负数时，则是length减去这个负数。

first()

作用：
获取匹配元素集合中第一个元素
用法：
1、不接受任何参数。
2、返回值包含了
3、如果找不到给定索引的元素，则返回一个空的jq对象。

last()

作用：
获取匹配元素集合中最后一个元素
用法：
1、用法与first一样。

filter()

作用：
筛选元素集合中匹配表达式 或 通过传递函数测试的 那些元素集合。
用法：
1、参数是一个选择器。返回的是匹配的一个集合。
2、参数也可以是一个函数，对于每个元素，如果函数返回true ，该元素将被包含在筛选集合中; 函数的参数是该元素在原先集合中的索引。

is( )

作用：
判断当前匹配的元素集合中的元素， 是否为一个选择器，DOM元素，或者jQuery对象， 如果这些元素至少一个匹配给定的参数，那么返回true
用法：
1、参数是一个选择器。返回的是匹配的一个布尔值。
2、参数也可以是一个函数，函数的返回值也就是is()的返回值。 

     $("ul").click(function(event) {

          var $target = $(event.target);

          if ( $target.is("li") ) {

            $target.css("background-color", "red");

          }

    });

map( )

作用：
通过一个函数匹配当前集合中的每个元素,产生一个包含新的jQuery对象。
用法：
1、该函数可以返回一个单独的数据或数据数组
2、与each()不同的是有一个返回值。

has( )

作用：
筛选匹配元素集合中的那些有（相匹配的选择器或DOM元素的后代元素）的集合中的元素。
用法：
1、常用于判断一个元素内部是否有另一个元素。
2、返回值是匹配的元素集合。

not( )

作用：
从匹配的元素集合中移除指定的元素。
用法：
1、参数可以是一个函数，如果函数的返回值是true，对应的元素将被踢出。
2、返回值是剩下元素的元素集合。 3、参数可以是选择器、DOM元素、jq对象。

slice( )

作用：
根据指定的下标范围，过滤匹配的元素集合，并生成一个新的 jQuery 对象。
用法：
1、用法与js原生的数组的slice方法类似。slice(1)、slice(2,4)

children( )

作用：
获得匹配元素集合中每个元素的子元素，选择器选择性筛选。
用法：
1、参数是一个选择器

closest( selector )

作用：
从元素本身开始，在DOM 树上逐级向上级元素匹配，并返回最先匹配的祖先元素。
用法：
1、参数是一个选择器，选择满足条件的父元素(包括自身)
2、第二个参数是指定查询的范围，这里的意思就是同事要满足是第二个参数的后代。
3、返回值是找到的父级。

find( selector )

作用：
通过一个选择器，jQuery对象，或元素过滤，得到当前匹配的元素集合中每个元素的后代。
用法：
1、$("#div").find("p")相当于$(" #div p ")

用法：

parent( )

作用：
取得匹配元素集合中，每个元素的直接父元素，可以提供一个可选的选择器。
用法：
1、跟选择器匹配的直接父元素才会选择。

parents( )

作用：
获得集合中每个匹配元素的祖先元素，可以提供一个可选的选择器作为参数。
用法：
1、跟选择器匹配的祖先元素才会选择。 2、返回的集合中包含的可能不止一个祖先元素。

parentsUntil( )

作用：
查找当前元素的所有的前辈元素，直到遇到选择器， DOM 节点或 jQuery 对象匹配的元素为止，但不包括这些元素。
用法：
1、跟nextUntil()的用法一样。

prev( )

作用：
获取集合中每个匹配元素紧邻的前一个兄弟元素。 如果提供了一种选择器，它将只检索匹配该选择器的紧邻的前一个兄弟元素。
用法：
1、跟next()的用法一样。

prevAll( )

作用：
获得集合中每个匹配元素的所有前面的兄弟元素，选择性筛选的选择器。
用法：
1、跟nextAll()的用法一样。

prevUntil( )

作用：
获取每个元素但不包括选择器，DOM节点，或者jQuery对象匹配的元素的所有前面的兄弟元素。
用法：
1、跟nextUntil( )的用法一样。

siblings( )

用法：

add( )

作用：
创建一个新的jQuery对象 ，元素添加到匹配的元素集合中。
用法： 1、已经存在的一个集合在添加一个元素。

end( )

作用：
终止在当前链的最新过滤操作，并返回匹配的元素的以前状态。
用法： 1、不接受参数
2、调用此函数将返回到最新一次的过滤操作之前状态。

        $('ul.first').find('.foo').css('background-color', 'red')
        .end().find('.bar').css('background-color', 'green');

addBack( )

作用：
添加堆栈中元素集合到当前集合，一个选择性的过滤选择器。
用法： 1、参数是一个选择器。
2、调用此函数将返回到最新一次的过滤操作之前状态。

        $('li.third-item').nextAll().addBack()
        .css('background-color', 'red');
        //原本不包括li.third-item，
        //但使用addBack()将堆栈中的集合加入到当前集合。

contents( )

作用：
获得匹配元素集合中每个元素的子元素，包括文字和注释节点。
用法： 1、这个方法不接受任何参数
2、调用此函数将返回到最新一次的过滤操作之前状态。
