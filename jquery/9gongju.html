函数操作

jQuery.proxy()

作用：
接受一个函数，然后返回一个新函数，并且这个新函数始终保持了特定的上下文语境。

用法：
1、本质是将this的指向改变，生成了一个新的函数。与call不同。 

        //这里的this是指#btn1
        $("#btn1").click(you.sayName);

        //这里的this是指you
        $("#btn2").click($.proxy(you.sayName,you));

数组和对象操作

jQuery.each()

作用：
迭代数组或对象。
用法;
1、第一个参数是一个数组或是一个对象，第二个参数是回调函数。
2、回调函数第一个参数是数组的索引或是对象的key,第二个是相应的value。

        var arr = ["A", "B", "C"];

        $.each(arr, function(index, val){

            alert(index +""+ val)
            //0A 1B 3C

        })

jQuery.extend()

作用：
将两个或更多对象的内容合并到第一个对象。
用法：
1、jQuery.extend( target [, object1 ] [, objectN ] )
将后面对象参数合并到第一个参数上。
2、jQuery.extend( [deep ], target, object1 [, objectN ] )
如果第一个参数是true,则是深拷贝，不支持第一个参数是false。
3、如果只有一个参数提供给$.extend()，这意味着目标参数被省略。
在这种情况下，jQuery对象本身被默认为目标对象。这样，我们可以在jQuery的命名空间下添加新的功能。
这对于插件开发者希望向 jQuery 中添加新函数时是很有用的。
4、目标对象（第一个参数）将被修改。
5、如果有相同属性，后面对象会重写。

        var obj1 = {
            a:"A",
            b:"B",
            c:{
                aaa:{
                    aaa:"aaaaaa"
                }
            }
        }

        var obj2 = {
            a:"AAAAA",
            d:"D",
            c:{
                aa:"AA",
                bb:"BB"
            }
        }

        var o1 = $.extend({}, obj1, obj2);
        var o2 = $.extend(true, {}, obj1, obj2);

        alert( JSON.stringify(o1) );
        //{"a":"AAAAA","b":"B","c":{"aa":"AA","bb":"BB"},"d":"D"}
        //重写C.没有遍历。

        alert( JSON.stringify(o2) );
        //{"a":"AAAAA","b":"B","c":{"aaa":{"aaa":"aaaaaa"},"aa":"AA","bb":"BB"},"d":"D"}

jQuery.grep()

作用：
查找满足过滤函数的数组元素。原始数组不受影响。
用法：
1、jQuery.grep( array, function [, invert ] )
第一个参数是一个数组
第二参数是回调函数，函数的第一个参数是value,第二个是对应的索引。this指向window。返回true则加入返回的新数组里。
第三个参数true可选，表示返回false的值的数组。 

        var arr = [1,2,3,4,5];

        var NEW  = $.grep(arr, function(val){
            return val%2 == 0;
        })

        alert(NEW);//2,4

jQuery.map()

作用：
将一个数组中的所有元素转换到另一个数组中。
在数组的每一个元素或对象上应用一个函数并将结果映射到一个新的数组中。
用法：
1、接受两个参数。
第一个参数：数组中元素或对象的值
第二个参数：是该元素在数组中的索引值或该对象的键。 


        var arr = [1, 2, 3, 4, 5];

        var newarr = $.map(arr, function(val){
            return 2*val;
        });

        alert(newarr)//2、4、6、8、10

jQuery.inArray()

作用：
在数组中查找指定值并返回它的索引（如果没有找到，则返回-1）。
用法：
jQuery.inArray( value, array [, fromIndex ] )
第一个参数： 要查找的值
第二个参数： 数组
第三个参数： 开始查找的位置 

        var arr = [1, 23, 55, 77];
        alert($.inArray(23, arr));//1
