attr( )

作用：操作元素的特性。

用法：
1、获取匹配的元素集合中的第一个元素的属性的值。
2、两个参数设置特性，attr(attrName,value)。
3、第二个参数也可以是一个函数。回调函数的第一的参数是索引，第二个参数是原来的值。

            $("li").attr("id" , function(index){

                return "li"+ index;

            }).each(function(){

                $(this).text($(this).attr("id"));

            });

prop( )

作用：
获取匹配的元素集中第一个元素的属性（property）值
用法：
1、获取第一个元素的指定属性的值。
2、设置属性值。

        $( "input[type='checkbox']" ).prop({
            26.  disabled: true
        27.});

removeProp( )

作用：
删除property属性。

addClass( )

作用:添加class。
用法：
1、可以一次性添加多个class。
2、参数可以是一个函数，参数是当前元素的索引，函数必须返回一个字符串。

removeClass( )

作用：
删除class。

toggleClass( )

作用：
有就删除，没有就添加。

html( )

作用：
获取集合中第一个匹配元素的HTML内容
用法：
1、没有参数获取html内容。
2、有参数时设置指定元素的内容。

text( )

作用：
得到匹配元素集合中每个元素的合并文本，包括他们的后代。
用法：
1、不能获取script（用html）和input、textarea（用val）的内容。
2、没参数获取文本内容。
3、有参数则是覆盖原有的内容。
4、不识别尖括号，会将尖括号转义。

val( )

作用：
要用于获取和设置表单元素的值。

css( )

作用：
要用于获取第一个匹配的元素的计算后的样式。
用法：
1、参数可以是一个样式或者是有多个样式组成数组。
2、不能获取复合样式的值，需要分开写。
3、设置样式可以是一个属性或者是一个json对象。
$("p").css({ "color": "#ff0011", "background": "blue" });
4、参数是一个回调函数时，第一个参数是索引，第二个参数是原来的值。 

         $(function(){
            var oDiv =  $("#div1");

            var style = oDiv.css(["width", "height", "backgroundColor"]);

            oDiv.text(style.width+","+style.height+","+style.backgroundColor);
        });
    
    $("div").click(function() {

        $(this).css({

          width: function(index, value) {

            return parseFloat(value) * 1.2;

          },

          height: function(index, value) {

            return parseFloat(value) * 1.2;

          }

        });

    });

offset()

作用：
在匹配的元素集合中，获取的第一个元素的当前坐标，坐标相对于文档。
用法：
1、返回一个包含top 和 left属性的对象 。
2、不接受任何参数。
3、不能获取display:none;元素的值，但可以获取visibility:hidden;的元素。
3、设置匹配的元素集合中每一个元素的坐标，参数是一个json， 坐标相对于文档，使用相对定位。

position(　）

<ｐ> 作用：
离该元素最近的而且被定位过的祖先元素的偏移距离。
用法：
与offset()一样。

scrollTop()

作用： 获取匹配的元素集合中第一个元素的当前垂直滚动条的位置或设置每个匹配元素的垂直滚动条位置。 用法： 1、滚动条垂直位置和滚动区隐藏区域的顶部高度像素值是相同的。 如果滚动条是在最顶部，或者这个元素没有可滚动的，那么这个数字是0。 2、没有参数，获取值，有参数设置值。

scrollLeft()

与scrollTop一样。

height()、width()

作用：
返回当前元素的内容的width、height。
用法：
1、与css()获取的height不同的是，height()获取的不带单位。
2、参数可以是100px, 50%, 或者 auto。默认的单位是px。

        $("#div3").click(function(){
            $(this).height(100);
        });
