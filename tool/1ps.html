工具栏快捷键要在英文状态下输入

控制视图快捷键

  1. 自由的移动psd图片

    按住空格，鼠标拖动

  2. 缩放图皮

    alt + 滚轮
    或 ctrl + + 、 ctrl + -

  3. 将图片缩放到100%

    ctrl+ alt + 0

  4. 工作区切换

    tab

  5. 一直后退操作

    ctrl + alt + z

控制内容快捷键

  1. 通过图片选中图层

    ctrl + 右键

  2. 复制图层

    移动模式下，alt + 鼠标拖动

  3. 删除图层

    选中要删除的图层，delete

  4. 合并多个图层

    选中要合并的图层，ctrl + e

  5. 选区内填充颜色

    alt + delete（前景色）
    ctrl + delete（背景色）

  6. 缩放当前图层

    ctrl + t

工具快捷键

  1. 画一个选区

    m

  2. 放弃选区

    ctrl + d

  3. 吸管工具

    i

  4. 文字工具

    t

  5. 切片工具

    c

  6. 缩放当前图层

    ctrl + t

图片格式

jpg： 色彩众多，不透明 ，内容性
png8：色彩简单，需求透明 ，修饰性
png24：半透明，在IE6下不兼容，解决存两份，一份是png8
