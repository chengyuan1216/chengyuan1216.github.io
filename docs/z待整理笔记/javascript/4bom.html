2、window.open()

window.open(URL,name,specs,repalce)
打开一个新的窗口（页面）。
1、至少有两个参数：新窗口的URL、打开的方式
URL：如果为空，则默认打开一个空白的页面。
name： _self---当前窗口打开
_blank---新窗口打开(默认)。等。。。。
specs: 新窗口属性列表。
--->width/height（尺寸）
--->top/left（离屏幕的距离）
--->menubar/toolbar/scrollbar/status（取值均为yes/no）
replace：
2、返回值：新打开窗口的window对象。
