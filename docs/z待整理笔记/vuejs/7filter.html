过滤器

作用：过滤模板数据,使用处理过的数据。model ->过滤 -> view。
用法：
{{msg| filterA}}
{{msg| filterA | filterB}}//使用多个过滤器
{{msg| filterA(参数)}}

系统提供一些过滤器:
uppercase eg: {{'welcome'| uppercase}}
lowercase
capitalize

currency 钱
....

数据配合使用过滤器:
limitBy 取几个 从哪开始
//只有一个参数时默认从第0个开始取 limitBy 4

filterBy
过滤数据,将包含指定内容的数据找出来 filterBy ‘谁’

orderBy 排序
orderBy 谁 1/-1
1 -> 正序
-1 -> 倒序

用法

        Vue.filter(name,function(input){

        //在创建vue实例对象前定义，需要过滤的数据作为函数的第一个参数

        });
    
        有问题，用不了
        双向过滤器:*
        Vue.filter('filterHtml',{

            read:function(input){                      //model-view
                return input.replace(/<[^<+]>/g,'');
            },
            write:function(val){                       //view -> model
                return val;
            }

        });

        数据 -> 视图
        model -> view

        view -> model

{{ num|double }}     {{ num2|before("￥") }}

        //将数据乘以2输出
        Vue.filter("double",function(input) {
            return input * 2;
        });

        //在数据强添加中指定的前缀输出
        Vue.filter("before",function(input,str) {
            return "" + str + input;
        });
