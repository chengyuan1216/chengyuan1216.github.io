show()

  1. 作用：
    • 显示隐藏的匹配元素。
  2. 用法：
    1. 没有参数：匹配的元素将被立即显示，没有动画。
    2. show( [duration ] [, complete ] )
      第一个参数：运动时间
      第二个参数：运动完后执行的函数。

hide()

  1. 作用：
    • 隐藏匹配的元素。
  2. 用法：
    1. 没有参数：匹配的元素将被立即隐藏，没有动画。
    2. hide( [duration ] [, complete ] )
      第一个参数：运动时间
      第二个参数：运动完后执行的函数。
    3. .hide()方法将为匹配元素的宽度，高度，以及不透明度，同时执行动画。 一旦透明度达到0，display样式属性将被设置为none

animate()

  1. 作用：
    • 自定义动画。
  2. 用法：
    1. animate( properties [, duration ] [, easing ] [, complete ] )
      properties : 要运动到的目标状态。
      duration ：运动时间。
      easing ：运动形式。
      complete ：运动完执行的函数。
    2. hide( [duration ] [, complete ] )
      第一个参数：运动时间
      第二个参数：运动完后执行的函数。
    3. 不能运动背景颜色。

stop()

  1. 作用：
    • 停止匹配元素当前正在运行的动画。
  2. 用法：
    1. stop()
      默认只会阻止当前运动。
      当前运动没有到目的地，而后续会继续运动到达目的地。
      相当于stop(false,false)
    2. stop(true)
      当前运动没到目的地，后续运动取消。
      相当于stop(true,false)
    3. stop(true，true)
      当前运动立即停止并且到达目的地，后续运动取消。
    4. 连个参数：
      第一个：是否取消以列队动画
      第二个;是否当前动画立即完成。
      默认均为false。

delay()

作用：
设置一个延时来推迟执行队列中后续的项。

finish()

作用：
停止当前正在运行的动画，删除所有排队的动画，并完成匹配元素所有的动画。
