DOM

el

vm实例对应的属性：vm.$el

vm实例对应的方法：vm.$mount()

  1. Vue 实例使用的根 DOM 元素。
  2. 提供的元素只能作为挂载点。所有的挂载元素不会被 Vue 生成的 DOM 替换。
  3. 如果 render 函数和 template 属性都不存在，挂载 DOM 元素的 HTML 会被提取出来用作模板，
  4. 使用模板的优先顺序是：render > template > innerHTMl

template

  1. 一个字符串模板作为 Vue 实例的标识使用。模板将会 替换 挂载的元素。挂载元素的内容都将被忽略，除非模板的内容有分发插槽。

数据

data

vm实例对应的属性：vm.$data

组件的data必须以一个函数返回一个json的形式定义

当一个 Vue 实例被创建时，它向 Vue 的响应式系统中加入了其 data 对象中能找到的所有的属性。 当这些属性的值发生改变时，视图将会产生“响应”，即匹配更新为新的值。

值得注意的是只有当实例被创建时 data 中存在的属性是响应式的。后来添加的不会更新视图。 

        var vm = new Vue({
            data:{
                a:12,
                b:13
            }
        })

        alert(vm.a === 12);//true

        alert(vm.data)//undefined

        alert(vm.$data.a)//12

vm.$watch(dataname,fnCb,{deep:true}); //深度监视 对象
