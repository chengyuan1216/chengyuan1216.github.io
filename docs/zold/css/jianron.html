IE兼容写法

通过条件注释语句，给html标签添加不同的class，然后就可以用class添加不同的样式。 

    <!DOCTYPE html>
    <!--[if IE 8 ]> <html class="ie8" lang="en"> <![endif]-->
    <!--[if IE 9 ]> <html class="ie9" lang="en"> <![endif]-->
    <!--[if (gt IE 9)|!(IE)]><!-->
    <html lang="en"> <!--<![endif]-->

8、IE6下块元素双边距问题
float + margin ,左右margin值会变成两倍(只有第一个的margin-left 和最后一个的margin-right会变化)。

11、IE6下的文字溢出问题
