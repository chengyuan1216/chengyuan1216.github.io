新增表单控件

email
电子邮箱文本框，界面和普通文本输入框一样。
作用：在提交表单的时候验证邮箱的格式是否正确。

tel
电话号码。pc端没有什么变化，移动端会切换到数字，没有验证。
url
网址。
作用：验证网址。网址必须带http://或https://。
search
搜索
作用：输入文本时，输入框带有一个‘X’，点击清除内容。
range
数值选择器。
作用：数值选择。
属性：
step:步值
min：最小值
max：最大值
value：当前值
number
数字选择。
作用：界面和文本输入框一样，只是多了两个按钮，可以调整数字的大小。
step:步值
min：最小值
max：最大值
value：当前值
color
颜色选择器
作用：点击弹出一个颜色选择框。

时间相关的控件

datetime
显示完整的时间。

datetime-local
显示完整年月日时间, 不含时区。
作用：点击弹出一个时间选择器
time
显示完整时间 不含时区
作用：和上面一样

date
只显示日期，不显示时间
week
显示‘2017年 第 20周’
month
显示‘2017年 09月’
