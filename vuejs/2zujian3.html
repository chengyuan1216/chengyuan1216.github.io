使用插槽分发内容

所谓的插槽就是，在子组件通过slot标签占一个位置，然后在父组件内使用子组件时，父组件可以将需要的DOM添加在插槽的位置。

父组件模板的内容在父组件作用域内编译；子组件模板的内容在子组件作用域内编译。

我进入插槽拉

我也进入插槽拉

我是导航

我是内容

我是底部

在使用插槽时一定要用div包裹住模板内的所有东西，否则不起作用。

 

        <div id="app">
            <div>
                <v-one>
                    <p>我进入插槽拉</p>
                    <p>我也进入插槽拉</p>
                </v-one>

                <v-two>
                    <p slot="nav">我是导航</p>
                    <p slot="main">我是内容</p>
                    <p slot="footer">我是底部</p>
                </v-two>


                <v-three>
                    <template scope="props">
                        <p>{{props.text}}</p>
                    </template>
                </v-three>

            </div>
        </div>


        <template id="one">
            <div>
                <h4>组件一：单插槽</h4>
                <slot>如果没有替换则使用默认值</slot>
            </div>
        </template>


        <!-- 具名slot -->
        <template id="two">
            <div>
                <h4>组件二：具名插槽</h4>
                <p>
                    插槽定义 ： name = "nav" <br/>
                    进入插槽的对象 ： slot = "nav"
                </p>
                <nav>
                    <slot name="nav"></slot>
                </nav>
                <main>
                    <slot name="main"></slot>
                </main>
                <footer>
                    <slot name="footer"></slot>
                </footer>
            </div>
        </template>

        <template id="three">
            <div>
                <h4>组件三：作用域插槽 </h4>
                <p>把数据传递给slot</p>
                <slot :text="three"></slot>
            </div>
        </template>



        Vue.component("my",{
            template:"#my"
        });

        new Vue({
            el: '#app',
            components: {
                'v-one': {
                    template: '#one'
                },
                'v-two': {
                    template: '#two'
                },
                'v-three': {
                    template: '#three',
                    data:function(){
                        return {
                            three : '我是组件传给进入插槽的对象的数据'
                        }
                    }
                }
            }
        });
