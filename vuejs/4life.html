1、什么是生命周期？

每个 Vue 实例在被创建之前都要经过一系列的初始化过程。

例如需要设置数据监听、编译模板、挂载实例到 DOM、 在数据变化时更新 DOM 等。

同时在这个过程中也会运行一些叫做生命周期钩子的函数， 给予用户机会在一些特定的场景下添加他们自己的代码。

2、Vue实例的4个阶段

实例创建

beforeCreate
created

实例挂载

beforeMount
mounted

1、将vue对象与一个html模板建立联系。

2、挂载的元素作为内容的输出位置。使用el或$mount()。 挂载完后寻找模板。 如果定义了模板，就使用定义的模板。 如果没有就将挂载的元素作为模板。

数据更新

beforeUpdate
updated

只有数据改变时 才会触发。

实例销毁

beforeDestroy
destroyed
不会主动销毁。

3、代码

    
<input id="app" v-model="app">

var vm = new Vue({
    el:'#app',

    data:{app:'Vue'},

    beforeCreate:function(){alert("实例创建前。");},
    created:function(){alert("实例创建完成。");},

    beforeMount:function(){alert("实例挂载前")},
    mounted:function(){alert("实例挂载完成")},

    beforeUpdate:function(){alert("数据更新前")},
    updated:function(){alert("数据更新后")},

    beforeDestroy:function(){alert("销毁前")},
    destroyed:function(){alert("销毁后")}
});

vm.$destroy();//销毁需要通过手动完成。
